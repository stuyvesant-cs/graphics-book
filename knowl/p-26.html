The multiplicative identiy matrix must:
  • Be a square matrix.
  • Have a diagonal of 1s, from upper-left to lower-right corners.
  • Have all other values be 0.
A \(4\times4\) identity matrix:
\begin{equation*} \begin{bmatrix}1 \amp 0 \amp 0 \amp 0 \\\ 0 \amp 1 \amp 0 \amp 0 \\\ 0 \amp 0 \amp 1 \amp 0 \\\ 0 \amp 0 \amp 0 \amp 1 \end{bmatrix} \end{equation*}
